### Setting

- **composer install (generate composer.lock file)**
- **Config .env**
- **docker compose up -d --build**
- **public port network (WSL)**
- **docker exec -it face_search bash**
- **php artisan key:generate**
- **chmod -R 0777 /public /storage /bootstrap**
